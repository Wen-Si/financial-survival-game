# 🎮 金融职场生存者游戏

AI 驱动的金融职场生存模拟游戏，玩家创建数字人角色后由 AI 自主操控在虚拟职场中生存发展。

## 🚀 快速开始

### 后端启动
```bash
cd backend
npm install
cp .env.example .env
# 编辑 .env 配置你的 MongoDB URI
npm run dev
```

### 前端启动
```bash
cd frontend
npm install
npm start
```

## 🎯 游戏特色

- **AI 驱动**: 使用 GLM-4.5-Flash 大模型驱动数字人决策
- **自然语言角色创建**: 用自然语言设定角色品格、性格、技能
- **100+ 场景**: 覆盖投行、商行、证券、基金、保险五大金融领域
- **多用户支持**: 支持多用户同时游戏，实例隔离
- **自主成长**: 角色在 AI 操控下自主生存、成长与发展

## 🏢 游戏场景

### 投行部门 (20 场景)
- IPO 项目组、并购重组部、债券承销部等

### 商业银行 (20 场景)
- 对公业务部、零售银行部、风险管理部等

### 证券公司 (20 场景)
- 研究所、自营部、资产管理部等

### 基金公司 (20 场景)
- 权益投资部、固定收益部、量化投资部等

### 保险公司 (20 场景)
- 精算部、核保部、投资管理部等

## 🛠️ 技术栈

- **前端**: React + TypeScript + TailwindCSS
- **后端**: Node.js + Express
- **数据库**: MongoDB
- **AI**: GLM-4.5-Flash (智谱 AI)

## 🎮 游戏说明

1. 注册/登录账号
2. 创建你的数字人角色（用自然语言描述）
3. 选择职业发展方向
4. AI 将自动操控角色在金融职场中生存
5. 随时查看角色状态、成长日志和职业发展
